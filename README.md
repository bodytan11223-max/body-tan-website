# Body Tan Website

Premium tanning and wellness website built with Next.js.

## Deploy to Vercel (Easiest)

### Option 1: Deploy with GitHub (Recommended)

1. **Create a GitHub account** (if you don't have one): https://github.com/signup

2. **Upload this project to GitHub**:
   - Go to https://github.com/new
   - Create a new repository called `body-tan-website`
   - Upload all files in this folder to that repository

3. **Connect to Vercel**:
   - Go to https://vercel.com
   - Click "New Project"
   - Select "Import from GitHub"
   - Find your `body-tan-website` repository
   - Click "Import"
   - Vercel will automatically detect Next.js configuration
   - Click "Deploy"

4. **You'll get a live URL** like `body-tan-website.vercel.app` instantly!

### Option 2: Direct Upload to Vercel (No GitHub)

1. Go to https://vercel.com/import
2. Click "Project from Git" → "Continue with GitHub" (create account if needed)
3. Or use Vercel's drag-and-drop upload feature
4. Select all files in this folder and upload
5. Deploy!

## Custom Domain (Optional)

1. Buy a domain (e.g., `bodytanbrooklyn.com`) from:
   - Vercel Domains
   - GoDaddy
   - Namecheap
   - Any registrar

2. In Vercel project settings → Domains → Add your domain

3. Follow Vercel's DNS configuration instructions

## Testing Locally (Optional)

If you want to test before deploying:

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Visit http://localhost:3000 in your browser
```
